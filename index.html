C#

C# is a language often used for video game development, as well as for developing software and apps for computers and phones. It's a great choice for someone with an interest in making software or games.

Python

Python is a language often used for developing software and managing, analyzing, or visualizing data. For anyone with an interest in machine learning, Python is the go-to language.

JavaScript

JavaScript is designed for functionality and interactivity in websites, and is a strong option for anyone with an interest in web development.

Lua

Lua is a language made for extending the functionality of applications. On top of creating software from scratch, it can be used to add scripts or plugins to software, orfor modding purposes in video games.
